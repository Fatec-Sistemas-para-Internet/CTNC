# CTNC

| :--- | :--- |
| Institutional website developed for **CTNC** — a company specializing in natural therapies and occupational health — as an applied extension project for the *Design Practices* class of the [Internet Systems](https://fatecrl.edu.br/cursos/sistemas-para-internet) degree program at [Fatec Baixada Santista Rubens Lara](https://fatecrl.edu.br). | Site institucional desenvolvido para a **CTNC** — empresa especializada em terapias naturais e saúde ocupacional — como projeto de curricularização da matéria de *Práticas de Design* do curso de [Sistemas para Internet](https://fatecrl.edu.br/cursos/sistemas-para-internet) da [Fatec Baixada Santista Rubens Lara](https://fatecrl.edu.br). |
| The website consists of four sections: the main page presenting the company and its corporate plans; a detailed plans page; a page for offered courses; and an institutional "About Us" page. It was built entirely with HTML and CSS, without any external frameworks or libraries. | O site é composto por quatro seções: a página principal com apresentação da empresa e seus planos corporativos; uma página de planos detalhados; uma página de cursos oferecidos; e uma página institucional "Sobre nós". Foi construído inteiramente com HTML e CSS, sem frameworks ou bibliotecas externas. |
